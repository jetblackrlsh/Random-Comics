# Key Item Descriptions

## Key Item: Alien Prison Rock

- First appears: Issue #1, "The Selfish Wish"
- Visual identifiers: massive meteor-like stone-metal object with cratered black surface, embedded alien containment glyphs, blue-white cracks, broken capsule cavities, and prison-cell architecture fused inside the rock.
- Continuity notes: Mistaken for a shooting star by Rann. It was an eternal exile prison for the galaxy's worst criminals. Its crash kills Rann and frees the prisoners across Earth.
- Reference image: `../reference-images/alien-prison-rock-key-item.png`

## Key Item: Flux Armor

- First appears: Issue #1, "The Selfish Wish"
- Visual identifiers: dark extradimensional battle armor, fitted but practical, glossy black and gunmetal plates, faint Limbo texture, glowing question marks on chest, shoulders, and gauntlets. The question marks change color whenever Rann's power shifts.
- Continuity notes: Given to Rann by Dark Angel when he is returned to life. It becomes Rann's hero costume and the visible symbol of unstable powers.
- Reference image: `../reference-images/flux-armor-key-item.png`
